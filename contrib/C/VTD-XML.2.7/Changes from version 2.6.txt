Changes from Version 2.7 (10/2009)

Expanded VTD-XML's Core API 
 
* VTDNav: toStringUpperCase, toStringLowerCase, contains(), endsWith(), startsWith()
   

Improved Xpath
 
* added the following XPath 2.0 functions: abs(), ends-with(), upper-case(), lower-case()
* added support for variable reference 
* significantly enhanced XPath syntax, checking error reporting (Special thanks to Mark Swanson)
* Internal performance tuning

Bug Fixes in VTDGen, VTDNav and AutoPilot

Code Enhancement: Removed large number of warnings
